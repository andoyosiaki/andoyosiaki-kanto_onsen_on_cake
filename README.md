## 簡単な説明
**<a href="https://2.kagome.xyz/">「関東温泉」</a>**  
をcakephpで作り変えました。bootstrapを使用していないので、細かなデザインに変更がある程度で機能は同じになります。

## 機能
1. コンテンツ投稿機能（管理ユーザーのみ)
1. コンテンツ一覧表示機能
1. コンテンツ詳細表示機能
1. コンテンツ検索機能
